# 🛒 Getir Clone

This is a cloned version of the Getir application, developed using React Native and Expo.

## 📱 Features

- ✅ Modern and responsive UI/UX
- ✅ Bottom Tab navigation
- ✅ Support for different device types (iOS, Android, Web)
- ✅ Redux integration with state management
- ✅ TypeScript support
- ✅ Gesture handling and safe area compatibility

## 🛠️ Technologies

- React Native
- Expo
- React Navigation
- Redux
- TypeScript
- React

## 📋 Requirements

- Node.js (v14 or higher)
- npm or yarn
- Expo CLI
